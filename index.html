ZMR 250.

This is a repository for a full featured cheap ZMR 250 build.

Features

The build includes the following

  • ZMR/QAV 250 cloned fiberglass frame
  • Replacement Carbon Fiber Arms
  • Power Distribution PCB
  • Naze 32 10DoF r5 running Cleanflight (1.1.0)
  • GPS (ubloxm 7)
  • 15 Amp Multistar ESC's
  • 4x DYS BE1806 2300KV Brushless Motor
  • 4x 5030 Gemfan Propellers (CW/CCW)
  • Current Sensor Module
  • Minim OSD (mini)
  • TS5823 5.8G 200mW 32CH AV Transmitter Module
  • Sony 960H CCD Effio-E DSP 700TVL Board Camera
